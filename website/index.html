About Freshtally

Freshtally is a smart retail management app that optimizes supermarket operations. Designed for managers, attendants, and customers, it streamlines shopping, inventory, promotions, and shelf tracking with real-time data and intelligent suggestions.

Features

Store Manager

  • Add new products
  • Input batch numbers and expiry dates
  • Enter supplier details
  • Manage user access and user codes

Manager

  • Create promotions
  • View and update product details
  • Track product closures
  • View item location analytics
  • Receive promotional suggestions

Customer

  • Search and browse products
  • View discounted and promotional products
  • Add to shopping list
  • Receive offer notifications

Shelf Attendant

  • Shelf mapping and product placement
  • Edit shelf locations
  • Get price change notifications
  • Receive shelf mapping suggestions
